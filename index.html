Hablidades

Lenguajes que frecuento Actualmente:

Tengo experiencia en el desarrollo web y movil, he trabajado con diversos frameworks que abarcan desde el frontend hasta el backend, incluyendo REACT.JS, FLUTTER, BOOTSTRAP, NODE.JS, LARAVEL y DJANGO. En el front-end mis conocimientos en FLUTTER me permiten el desarrollo de aplicaciones moviles multiplataforma con interfaces intuitivas, asimismo, en el ámbito web, he trabajado con React para construir interfaces interactivas y dinámicas, así como con Bootstrap para lograr diseños modernos y receptivos. En el backend, mi destreza se extiende a NODE.JS para entornos de servidor eficientes y LARAVEL, y DJANGO para el desarrollo robusto de aplicaciones. Mi comodidad con lenguajes como Python y JS se refleja en mi experiencia en (POO), (EDD) y (FP). Además, cuento con sólidos conocimientos en el diseño del modelo Entidad-Relacion y su implementacion, utilizando manejadores de BD SQL como PostgreSQL, MySQL y SQL Server. He aplicado estos conocimientos para respaldar aplicaciones de manera eficiente y segura.
